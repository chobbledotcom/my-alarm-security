We recently had a call from a customer who had moved in to a house in Sidcup a few weeks earlier. The existing Texecom alarm system had started bleeping, had error messages displayed on the panel and would no longer let the customer set the system. We attended the property and discovered that the system was still configured as a monitored alarm linked to the previous homeowner. We disconnected the old monitoring setup as the new homeowner did not want this set up. We reprogrammed the system for the new homeowner and set it up to suit their requirements. As the system was no longer going to be a monitored system, it would now be an audible only system which means that in the event of a break in - or attempted break in, the alarm would sound loudly. Everything sorted in just one callout with a good review left by the customer.
