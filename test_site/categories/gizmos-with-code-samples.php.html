Programmable Gizmos

Gizmos that can be controlled via API or scripting.

API Example

Control your gizmo with simple code:


const gizmo = new Gizmo({
  id: "gizmo-123",
  mode: "turbo"
});

gizmo.activate();

Configuration

JSON configuration example:


{
  "gizmo": {
    "name": "My Gizmo",
    "settings": {
      "speed": 100,
      "mode": "auto"
    }
  }
}

Shell Commands

Control via command line:


$ gizmo-cli --mode turbo --speed 100
$ gizmo-cli --status
$ gizmo-cli --shutdown

HTML in Content

Example of embedding: <gizmo-widget id="test"></gizmo-widget>
